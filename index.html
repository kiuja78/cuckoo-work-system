System Overview

시스템 소개

쿠쿠 업무 자동화

반복적인 고객 관리, 계정 정리, 파일 변환, 방문 준비 업무를 더 빠르고 일관성 있게 처리할 수 있도록 지원합니다.

고객 방문 업무 지원

고객명, 주소, 담당자 정보를 기준으로 방문 전 확인이 필요한 정보를 정리하고 현장 업무 준비 시간을 줄입니다.

다운로드와 업데이트

최신 프로그램 파일과 업데이트 내역을 제공하여 사용자가 항상 최신 버전으로 업무를 진행할 수 있도록 안내합니다.

Automation Scope

업무 자동화 지원 범위

파일 정리

엑셀, PDF 등 업무 파일을 기준에 맞게 정리하고 필요한 항목을 빠르게 확인할 수 있도록 지원합니다.

고객 데이터 확인

고객번호, 고객명, 연락처, 주소, 상품 정보 등 현장에서 필요한 데이터를 보기 쉽게 정리하는 방향으로 확장할 수 있습니다.

업데이트 관리

기능 개선, 오류 수정, 신규 버전 배포 정보를 한 곳에서 확인하고 최신 파일을 다운로드할 수 있도록 구성했습니다.

Customer Visit Map

고객 방문지도 제공

카카오맵 API를 활용하여 고객 주소를 지도 위에 표시하고, 고객명 또는 담당자 기준으로 방문 위치를 확인할 수 있도록 준비된 영역입니다. 카카오 Developers에서 JavaScript 키를 발급받은 뒤 index.html의 '40c9847ec2514785762cf5e4ac387eea' 값을 실제 키로 교체하면 지도 기능을 연결할 수 있습니다.

  • 고객 주소 기반 위치 확인
  • 고객명 마커 표시
  • 방문 담당자별 동선 확인
  • 엑셀 업로드 지도 표시 기능 확장 가능

Guide

사용 방법

1

상단 메뉴에서 다운로드를 선택합니다.

2

필요한 프로그램의 다운로드 버튼을 눌러 ZIP 파일을 내려받습니다.

3

ZIP 파일인 경우 압축을 해제한 뒤 실행 파일을 실행합니다.

4

GitHub Releases와 연동된 최신 버전 정보를 확인하여 항상 최신 파일로 업무를 진행합니다.
