About

서비스 소개

방문 업무 효율화

고객 주소를 기준으로 방문 위치를 확인하고, 담당자의 이동 동선과 현장 업무 준비를 돕습니다.

카카오맵 기반 위치 확인

카카오맵 API를 활용하여 고객 주소를 지도 위에 표시하고, 고객명 기준으로 마커를 확인할 수 있습니다.

업데이트 파일 배포

업무자동화 프로그램의 최신 버전, 수정 내역, 설치 파일을 한 곳에서 안내합니다.

Kakao Map

카카오맵 고객 방문지도

카카오 Developers에서 JavaScript 키를 발급받은 뒤, index.html 파일의 KAKAO_JAVASCRIPT_KEY 값을 실제 키로 교체하면 지도 기능을 연결할 수 있습니다.

  • 고객 주소 기반 위치 확인
  • 고객명 마커 표시
  • 방문 담당자 동선 확인
  • 엑셀 업로드 기능 확장 가능

Guide

사용 방법

1

다운로드 버튼을 눌러 최신 프로그램 파일을 내려받습니다.

2

ZIP 파일인 경우 압축을 해제한 뒤 실행 파일을 실행합니다.

3

사내 보안 경고가 뜨면 관리자 또는 담당자에게 파일 출처를 확인합니다.

4

업데이트 공지를 확인하여 최신 버전을 유지합니다.
