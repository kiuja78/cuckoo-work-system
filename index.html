Overview

홈페이지를 단일 다운로드 페이지에서 통합 시스템 허브로 확장했습니다

필요한 시스템만 선택해서 내려받을 수 있고, 최신 업데이트를 지원합니다. 복잡한 설명보다 실제 현장 사용 흐름이 쉽게 보이도록 심플하고 직관적인 구조로 재구성했습니다.

01

프로그램 통합 안내

지금까지 제작된 주요 시스템을 한 페이지에서 구조적으로 정리해, 어떤 프로그램이 어떤 목적에 쓰이는지 빠르게 파악할 수 있습니다.

02

선택형 다운로드

업무자동화시스템부터 수당계산시스템까지 필요한 시스템만 선택해 내려받을 수 있도록 구성했습니다.

03

업데이트 관리 단순화

GitHub Releases와 연결해 다운로드 파일을 안정적으로 관리하고, 프로그램별 배포 파일을 한 번에 안내합니다.

System Line-up

구성 프로그램

각 프로그램의 역할을 쉽게 이해할 수 있도록 핵심 기능 중심으로 정리했습니다.

Main Program Automation 파일 정리 · 방문 준비 · 지도 확장

업무자동화시스템

고객정보 파일 정리, 계정 관리, 방문 준비, 지도 연동 기반 기능을 위한 메인 자동화 시스템입니다.

  • PDF/엑셀 자동 정리
  • 고객 방문 리스트 지원
  • 현장 업무 전처리 최적화
Sales Flow Sales Manager 접수 · 멤버십 · 영업현황

영업관리시스템

영업현황, 접수내역, 멤버십 내역, 컨택 관리 등 영업 흐름 전체를 관리하는 시스템입니다.

  • 영업현황 및 목표월 관리
  • 접수/멤버십 데이터 관리
  • 필터/검색 기반 운영
Quotation Quote Auto 제품 선택 · 가격 · 견적 출력

견적자동화시스템

제품 선택, 가격 정보, 대표 이미지, 견적서 출력 흐름을 자동화하는 견적 지원 시스템입니다.

  • 제품별 견적 생성
  • 대표 이미지 연동
  • 견적서 작성 시간 단축
Mobile Customer Care 모바일 확인 · 위치 · 응대 지원

고객관리모바일시스템

모바일 환경에서 고객정보, 방문 위치, 연락 흐름을 빠르게 확인할 수 있도록 구성한 고객관리 지원 시스템입니다.

  • 모바일 화면 최적화
  • 방문 위치 및 고객 확인
  • 현장 응대 편의성 강화
Calculator Price & Commission 제품가격 · 수당 · 조건 계산

제품가격수당계산시스템

제품군, 모델, 약정, 관리방식에 따른 가격과 수당 정보를 빠르게 확인하기 위한 계산 시스템입니다.

  • 제품별 가격 조회
  • 조건별 수당 계산
  • 현장 영업용 빠른 확인
