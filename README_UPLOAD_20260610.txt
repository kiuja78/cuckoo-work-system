반영 내용
1. 제품가격수당계산시스템 실행 주소를 새 주소로 변경
   - https://bit.ly/4uf5cRE

2. 현재 홈페이지 기본 버전값 정리
   - 업무자동화시스템: V12.70
   - 영업관리시스템: V8.90
   - 제품가격수당계산시스템: V79

3. 제품가격수당계산시스템은 ZIP 다운로드가 아니라 웹앱 바로가기 방식 유지

업로드 방법
1. 이 ZIP 파일을 압축 해제합니다.
2. github_pages_update_site 폴더 안의 파일을 GitHub 저장소 최상위에 업로드합니다.
   - assets/site-config.js 는 기존 assets/site-config.js 에 덮어쓰기
   - version.json 은 기존 version.json 에 덮어쓰기
3. 커밋 메시지 예시:
   제품가격수당계산시스템 링크 변경 및 버전 기본값 업데이트
4. 홈페이지 접속 후 Ctrl + F5 로 강력 새로고침합니다.

중요: 자동 버전 반영 규칙
현재 홈페이지 app.js는 GitHub Releases 최신 릴리즈의 제목/본문에서 "시스템명+V버전" 형식을 읽어 버전을 자동 표시하는 구조입니다.
제품가격수당계산시스템 버전도 자동으로 바뀌게 하려면, 새 릴리즈 제목 또는 본문에 아래처럼 꼭 포함해 주세요.

예시 릴리즈 제목:
쿠쿠업무자동화시스템 V12.70 / 지국영업관리시스템 V8.90 / 제품가격수당계산시스템 V79

예시 릴리즈 본문:
제품가격수당계산시스템 V79
제품&수당계산시스템 V79
제품가격 수당계산 시스템 V79

주의
- 제품가격수당계산시스템 웹앱 자체가 별도 버전 API를 제공하지 않으면, 홈페이지가 웹앱 내부 버전을 직접 읽어올 수 없습니다.
- 그래서 현재 구조에서는 GitHub Releases 제목/본문에 버전을 적는 방식이 가장 안전한 자동 반영 방식입니다.
